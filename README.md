## Econometría con Python - Material de apoyo

### Cómo usar este repositorio

Este repositorio contiene los notebooks y materiales de apoyo para el curso de Econometría con Python. Cada sesión del curso tiene su propio notebook.

### Cómo configurar cada notebook

Tras abrir los enlaces que se muestran más abajo en Google Colab, es necesario ejecutar la primera celda de cada notebook para configurar el entorno de trabajo. Esto es automático, pero puede tardar unos segundos. Una vez la primera celda haya terminado de ejecutarse, ya se pueden ejecutar las demás celdas del notebook.
Para cada notebook se debe repetir este proceso.
